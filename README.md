# Conversor de Bases Numéricas 

Este projeto é um trabalho da disciplina de Arquitetura e Organização de Computadores.
O sistema tem como objetivo permitir que o usuário selecione duas bases numéricas e realize a conversão de um número inteiro entre elas.

## Objetivo

Implementar um conversor capaz de:
- Selecionar uma base de origem
- Selecionar uma base de destino
- Converter um número inteiro entre essas bases
